# Employee Performance and Salary Prediction

A Machine Learning and Data Analytics project built using Streamlit and Python to analyze employee performance and predict salary based on HR analytics data.

## 🚀 Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📌 Features
- Data Cleaning & Preprocessing
- Performance Score Analysis
- Outlier Detection
- Data Visualization
- Salary Prediction using Linear Regression

- ## 📈 Performance Score Calculation
Calculated performance score and categorized employees into:
- High Performance
- Average Performance
- Low Performance
